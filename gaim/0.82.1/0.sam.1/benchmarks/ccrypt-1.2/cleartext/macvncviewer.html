VNC Viewer for Macintosh

Introduction:

Use of the Mac viewer should be straightforward if you are familiar with the concepts behind VNC, and have a suitable VNC server running on another machine.  Read 'What is VNC?' and 'Getting Started' on the VNC web site if you haven't already done so - they'll give you a good overview, despite being Unix and Windows-orientated.

Documentation

Full documentation is now included with the package. Please refer to this for further details.

Requirements

To use the Macintosh VNC viewer, you will need:

  • MacOS 7.1 or greater. System 7.5.3 is available as a free download from Apple.
  • Open Transport (1.1.1 or later). OT 1.1.2 can be downloaded from Apple. If you are using MacOS later than 8.0 you can use 1.3

Any comments regarding this program should be directed to the VNC mailing lists.
